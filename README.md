# Should I Have a Cup of Coffee?

A single-page interactive web app built with vanilla HTML, CSS, and JavaScript.

## Features

- Mouse-driven parallax effect across layered watercolor artwork
- Canvas-based pixel transparency detection for precise click handling
- Particle system generating procedural steam animations
- Time-aware response logic
- CSS keyframe animations with randomized selection
- LocalStorage persistence
- Responsive layout with viewport-relative typography

## Stack

No frameworks or build tools. Just static files served as-is.
