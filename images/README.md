# Watercolor Cafe Images

Place your AI-generated watercolor cafe images here:

## Required Files

- `background.png` - Background layer (1920x1080px)
- `midground.png` - Middle layer (1920x1080px)
- `foreground.png` - Foreground layer (1920x1080px)
- `coffee-cup.png` - Hero coffee cup layer (1920x1080px)

## Style Guide

- **Format:** PNG with transparency
- **Style:** Watercolor painting aesthetic
- **Colors:** Warm tones (browns, creams, soft oranges, muted yellows)
- **Consistency:** Use same artistic style across all layers

Once you've added your images, the app will automatically use them for the parallax effect!
